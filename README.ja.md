# マップアプリ

このリポジトリには、CSV形式のデータを使ってマーカーを地図上に表示するシンプルなマップアプリケーションが含まれています。

## デモ
[テストマップ](https://code4fukui.github.io/map/)を見てみてください。[解説記事](https://fukuno.jig.jp/3277)もあります。

## 機能
- CSVデータから地図上にマーカーを表示する
- マーカーをクリックするとポップアップが表示される

## 使い方
HTMLファイルに`<csv-map>` タグを記述するだけで地図を表示できます。
